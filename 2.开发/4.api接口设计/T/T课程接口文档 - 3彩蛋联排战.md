# T课程接口文档-彩蛋联排战

### gameType = 3

1、 显示方式类型（showType）参数说明

|值  | 说明 |
| --- | --- |
| 0 | 单词 |
| 1 | 图片|

2、 返回参数说明

|名称  | 类型 | 说明 |
| --- | --- | --- |
| queContentId | long | 游戏问题内容项ID |
| showType | integer | 显示方式 |
| itmeElement | string | 单词或语句 |
| itmeImage | string | 对应图片 |
| itmeAudio | string | 对应音频 |
